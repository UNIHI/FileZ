# Upload-Dialog

* Datei: Auswahl der Datei
* Dauer: Wie lange soll die Datei verfügbar sein, beginnend ab Startdatum
* Startdatum: Ab diesem Datum wird die Datei verfügbar sein.
* Kommentar: Beliebiger Text, wird bei der Dateivorschau angezeigt.
* Datei einem Ordner zuweisen (optional): Die Datei wird einem Ordner zugewiesen. Es besteht die Möglichkeit, einem Ordner zugewiesene Dateien gesammelt als Liste anzuzeigen, indem die URL /'benutzername'/list/'ordnername' verwendet wird, wobei 'benutzername' durch den RZ-Benutzernamen und 'ordnername' durch den konkreten, vergebenen Ordnernamen zu ersetzen sind.
* Schicken Sie mir Benachrichtigungen per E-Mail (optional): Diese Option dient der Bestätigung über den erfolgreichen Dateiupload und der rechtzeitigen Benachrichtung über den Zeitpunkt, ab dem die Datei nciht mehr verfügbar und gelöscht sein wird.
* Passwort verwenden (optional): Um die Datei heruntenladen zu können, muss das auf der Vorschauseite eingegebene Kennwort mit dem hier angegebenen Kennwort übereinstimmen.
* Erfordere Login (optional): Der Benutzer muss eingeloggt sein, um die Datei herunteladen zu können.
  * Anmerkung zu den beiden vorhergehenden Optionen: Sie können in beliebiger Kombination verwendet werden. Eine Systemoption kann aber verlangen, dass mindestens eine der Zugriffsbeschränkungen aktiviert ist.
* Benutzervereinbarung: Nicht optional, falls das System es verlangt, dass der Benutzer dieser zustimmen muss. Durch das Setzen des Hakens wird der Vereinbahrung zugestimmt.

# Vorschau

Je nach Systemeinstellung startet der Download automatisch oder muss manuell durch Klick auf den dicken blauen awesome button gestartet werden.
Angezeigt wird:

* Verfügbarkeitszeitraum der Datei
* Vor- und Name des Uploaders (kein Nickname!)
* Option eine Datei dem Administrator zu melden. Gründe sind: Kopierrecht-Verletzung, Offensive Inhalte, Seitenregelverletzung. Es kann optional ein Kommentar hinterlassen werden.
